# Spring MVC 패턴을 이용한 중고 거래 웹 어플리케이션

OOP와 JDBC를 활용한 1차 프로젝트의 동일 한 주제를 Spring MVC 패턴을 활용해서 구현해보고 싶었으며,  1차 프로젝트에서 구현하는데 한계가 있었던 기능들을 이번 프로젝트에서 구현해보면서 다양하게 시도해 보고싶었습니다.
